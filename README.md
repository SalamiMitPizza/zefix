# ng2 boilerplate

Utilizes:
- Angular 2
- VCL
- webpack 2.1
- webpack dev server 2.1
- ng-vcl
- tslint


## Installation

```sh
npm install
```

## Development

```sh
npm run dev
```

## Building

```sh
npm run build
```
